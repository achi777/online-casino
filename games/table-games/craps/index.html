How to Play Craps

  • Pass Line: Win on 7 or 11 (Come Out). Lose on 2, 3, 12. If point is set, win if point is rolled before 7.
  • Don't Pass: Opposite of Pass Line. Win on 2 or 3, push on 12 (Come Out).
  • Come/Don't Come: Same as Pass/Don't Pass but placed during Point phase.
  • Field: One-roll bet. Win on 2, 3, 4, 9, 10, 11, 12. (2 and 12 pay double)
  • Any 7: One-roll bet. Win if total is 7. Pays 4:1
  • Any Craps: One-roll bet. Win on 2, 3, or 12. Pays 7:1
  • Hardways: Win if pair is rolled before 7 or easy way (e.g., Hard 8 = 4+4). Pays 7:1 or 9:1
